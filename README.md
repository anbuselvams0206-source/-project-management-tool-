# Project Management Tool

## Overview
A full-stack Project Management Tool similar to Trello or Asana.

## Features
- User Registration & Login
- JWT Authentication
- Create Projects
- Manage Tasks
- Assign Tasks
- Task Comments
- MongoDB Database
- REST API
- Responsive Frontend

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

## Installation

1. Clone the repository
2. Run:
   npm install
3. Create a `.env` file using `.env.example`
4. Start the server:
   npm start

## Project Structure

client/
- index.html
- style.css
- script.js

server/
- models/
- routes/
